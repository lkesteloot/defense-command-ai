# Defense Command AI

This is an attempt to write an AI to play the
TRS-80 video game Defense Command. The goal is
to win the [TRS8BIT 2024 contest](https://trs-80.org.uk/page5.html).

# Running

```sh
% pip install keras torch
% python dqn.py
```
